## VISUAL STYLE GUIDE

### Colors
- **Primary:** #FFDF00 (Golden Yellow)
- **Secondary:** #FF9500 (Orange)
- **Accent:** #FF7700 (Deep Orange)
- **Background:** #0D0D0D (Deep Black)
- **Text:** #FFFFFF (White)

### Typography
- **Headlines:** Bold, sans-serif (Inter/Geist)
- **Body:** Clean, modern weight
- **Numbers:** Monospace for data

### Transitions
- Smooth fades with golden particle effects
- Zoom transitions on feature reveals
- Glitch effects for market chaos scenes
- Light ray transitions between scenes

### Motion Graphics
- Floating data particles
- Pulsing glow effects on important elements
- Smooth camera movements (dolly, pan, zoom)
- Parallax depth on UI elements

---

## AUDIO GUIDE

### Music Style
- Epic orchestral (strings, brass, percussion)
- Electronic undertones (synth bass, arpeggios)
- Build-drop structure
- Reference: Hans Zimmer meets Daft Punk

### Sound Effects
- Deep bass impacts for transitions
- Data whoosh sounds
- Subtle UI clicks and beeps
- Heartbeat pulse during tension moments

### Recommended Tracks (Royalty-Free)
- Epidemic Sound: "Epic Cinematic" category
- Artlist: "Trailer" or "Corporate Epic"
- Envato Elements: "Cinematic Trailer" pack

---

## PRODUCTION NOTES

### Software Options
1. **After Effects** - Best for motion graphics
2. **Premiere Pro** - Video editing
3. **Cinema 4D** - 3D logo animation
4. **Figma** - UI mockup animations
5. **Blender** - Free 3D alternative

### Screen Recording
- Use high FPS (60fps) for smooth UI
- Record at 4K if possible
- Add motion blur in post

### Export Settings
- 1920x1080 or 3840x2160
- 60fps
- H.264 codec
- High bitrate (20-50 Mbps)
