## Market Data APIs

### GET /api/funding

Aggregates perpetual futures funding rates from 30 exchanges.

**Parameters**
| Param | Type | Default | Description |
|-------|------|---------|-------------|
| `assetClass` | string | `crypto` | Filter: `crypto`, `stocks`, `forex` |

**Response** (200)
```json
{
  "data": [
    {
      "symbol": "BTC",
      "exchange": "Binance",
      "fundingRate": 0.01,
      "fundingInterval": "8h",
      "markPrice": 67432.5,
      "indexPrice": 67428.3,
      "nextFundingTime": 1711584000000,
      "predictedRate": 0.0085,
      "type": "cex",
      "assetClass": "crypto",
      "marginType": "linear"
    }
  ],
  "health": [
    {
      "name": "Binance",
      "status": "ok",
      "count": 342,
      "latencyMs": 234,
      "error": null
    }
  ],
  "meta": {
    "totalExchanges": 30,
    "activeExchanges": 28,
    "totalEntries": 4521,
    "timestamp": 1711584000000
  }
}
```

**Headers**
- `X-Cache: HIT|MISS|FILTERED|STALE` — Cache status
- `Cache-Control: public, s-maxage=15, stale-while-revalidate=30` (on MISS)
- `Cache-Control: public, s-maxage=10, stale-while-revalidate=20` (on STALE)

**Caching**: L1 2-min (raw) + per-assetClass response cache. Circuit breaker per exchange.

### GET /api/tickers

Aggregates 24h ticker data from 28 exchanges.

**Parameters**
| Param | Type | Default | Description |
|-------|------|---------|-------------|
| `symbols` | string | all | Comma-separated symbols (BTC,ETH) |

**Response** (200)
```json
{
  "data": [
    {
      "symbol": "BTC",
      "exchange": "Binance",
      "lastPrice": 67432.5,
      "quoteVolume24h": 12345678901,
      "baseVolume24h": 183245,
      "priceChangePercent24h": 2.34,
      "fetchedAt": 1711584000000
    }
  ],
  "meta": {
    "totalExchanges": 28,
    "activeExchanges": 26,
    "totalEntries": 3800,
    "timestamp": 1711584000000
  }
}
```

**Caching**: L1 3s, HTTP `s-maxage=2, stale-while-revalidate=5`. Deduped via `dedupedFetch()`.

**Aggregation**: Per-symbol highest volume exchange wins. Volume capped at $100B per exchange per symbol.

### GET /api/spreads/current

Real-time inter-exchange price spreads.

**Parameters**: None

**Response** (200)
```json
{
  "data": [
    {
      "symbol": "BTC",
      "spreadUsd": 12.3,
      "spreadPct": 0.018,
      "highExchange": "Bybit",
      "highPrice": 67445.3,
      "lowExchange": "Binance",
      "lowPrice": 67433.0,
      "exchangeCount": 14
    }
  ],
  "count": 245,
  "ts": 1711584000000
}
```

**Logic**: Groups tickers by symbol, filters outliers >5% from median, requires >=2 exchanges, sorts by spread % descending.

**Caching**: 30s in-memory. Max duration: 30s.

### GET /api/rsi

RSI-14 across 50 hardcoded symbols.

**Parameters**: None

**Response** (200)
```json
{
  "data": [
    {
      "symbol": "BTC",
      "rsi1h": 54.2,

      "rsi1d": 48.7,
```

## Authentication Routes

### POST /api/auth/signup

**Body**
```json
{ "name": "John", "email": "john@example.com", "password": "SecureP4ss!" }
```

**Validation**: Email format, password 8+ chars with 1 upper, 1 lower, 1 digit.
**Rate Limit**: 5 attempts / 15 min per IP.
**Response**: `{ "requiresVerification": true }`
**Side Effects**: Creates unverified user, generates 6-digit code (15min TTL), sends email via Resend.

### POST /api/auth/check-credentials

Login step 1 — validates without issuing session.

**Body**: `{ "email": "...", "password": "..." }`
**Rate Limit**: 10 attempts / 15 min per email (10K key cap).
**Response**: `{ "valid": true, "emailVerified": true, "requires2FA": true, "methods": ["totp", "email"] }`

### POST /api/auth/2fa/validate

Login step 2 — validates 2FA code.

**Body**: `{ "email": "...", "code": "123456", "method": "totp" }`
**Methods**: totp (OTPAuth +-30s window), email (atomic claim), backup (timing-safe compare + array_remove).
**Rate Limit**: 5 attempts / 15 min per email.
**Response**: `{ "valid": true }`

### POST /api/auth/2fa/setup

Enable TOTP 2FA (auth required).

**Response**: `{ "qrCode": "data:image/png;base64,...", "secret": "JBSWY3DPEHPK3PXP", "backupCodes": ["A1B2C3D4", ...] }`
**Details**: 20-byte secret, 6 digits, 30s period, SHA1. 8 backup codes (4 bytes each).

## Cron Jobs

All require: `Authorization: Bearer ${CRON_SECRET}`

### GET /api/cron/snapshot
**Schedule**: Every minute (processes every 10 min for full run)
**Logic**: Fetches /api/funding + /api/openinterest (25s timeout), picks top 200 symbols, saves to DB, computes spreads, prunes >30 day data.

### GET /api/cron/alerts
**Schedule**: Every 5 minutes
**Logic**: Fetches market data, checks user alert rules, sends email/Telegram/push, logs to alert_notifications, prunes >7 day notifications.

### GET /api/cron/arbitrage-alerts
**Schedule**: Every 5 minutes
**Logic**: Detects price/funding arbs above threshold, sends Telegram to subscribers with cooldown.

### GET /api/cron/liquidation-ingest
**Schedule**: Every minute
**Logic**: Polls Binance (`/fapi/v1/allForceOrders`), OKX (per-ULY, 2 req/2s), HTX (batch 4), gTrade (24h history filter), Deribit (filter liquidation=true). Dedup via unique index.

### GET /api/cron/portfolio-snapshot
**Schedule**: 12:00 UTC daily
**Logic**: Snapshots all user portfolio values with current prices.

### GET /api/cron/telegram-reports
**Schedule**: 08:00 UTC daily, 10:00 UTC Sunday (weekly)
**Logic**: Sends market summary reports to subscribed Telegram users.
