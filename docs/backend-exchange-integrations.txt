### 1. Binance

**Type**: CEX | **Data**: Funding, OI, Tickers, Spot | **Status**: Active

| Endpoint | URL | Method |
|----------|-----|--------|
| Funding (Linear) | `https://fapi.binance.com/fapi/v1/premiumIndex` | GET |
| Funding (Inverse) | `https://dapi.binance.com/dapi/v1/premiumIndex` | GET |
| Tickers | `https://fapi.binance.com/fapi/v1/ticker/24hr` | GET |
| OI (per-symbol) | `https://fapi.binance.com/fapi/v1/openInterest?symbol={symbol}` | GET |
| Spot Tickers | `https://api.binance.com/api/v3/ticker/24hr` | GET |
| Klines | `https://fapi.binance.com/fapi/v1/klines` | GET |
| Liquidations | `https://fapi.binance.com/fapi/v1/allForceOrders?limit=200` | GET |
| Orderbook | `https://fapi.binance.com/fapi/v1/depth` | GET |
| Agg Trades | `https://fapi.binance.com/fapi/v1/aggTrades` | GET |
| L/S Ratio | `https://fapi.binance.com/futures/data/globalLongShortAccountRatio` | GET |
| WebSocket | `wss://fstream.binance.com/ws/{pair}@aggTrade` | WS |

**Symbol Format**: `BTCUSDT` → strip `USDT` = `BTC`
**Precision**: Funding rates as decimals (multiply by 100 for %). Volumes as floats.
**Funding Interval**: 8h
**Fallback Domains**: `fapi.binance.me`, `fapi1.binance.com`, `fapi2.binance.com` + proxy
**Auth**: None required for public endpoints
**Rate Limits**: OI batched at 50 symbols per request
**Special**: Two separate fetchers for USDT-M (linear) and COIN-M (inverse) perpetuals

### 23. GMX V2

**Type**: DEX (Arbitrum) | **Data**: Funding, OI (NO Tickers) | **Status**: Active

| Endpoint | URL | Method |
|----------|-----|--------|
| Markets Info | `https://arbitrum-api.gmxinfra.io/markets/info` | GET |

**Symbol Format**: `BTC/USD [WBTC.b-USDC]` → `name.split('/')[0]` = `BTC`. Strip `.v\d+` suffix (XAUT.v2 → XAUT).
**Precision**: ALL values are BigInt strings at 1e30 precision. MUST use `BigInt()` constructor.

**Funding Formula**:
```javascript
// Annual rate at 1e30 → hourly percentage
baseRate = Math.min(Math.abs(fundingRateLong), Math.abs(fundingRateShort))
hourlyRate = Number(BigInt(baseRate)) / 1e30 / 8760 * 100

// Direction: fundingRateLong < 0 → longs pay shorts
```

**OI Formula**:
```javascript
oiUsd = Number(BigInt(openInterestLong)) / 1e30  // Already in USD
```

**Funding Interval**: 1h (continuous accrual every second, reported as 1h)
**Special**: Deduplicates by symbol (multiple collateral pools) — keeps highest OI pool. Multiple BTC markets: [WBTC.b-WBTC.b], [WBTC.b-USDC], [tBTC-tBTC].
**Known API Changes (Mar 2026)**:
- `longOpenInterest` → `openInterestLong`
- `shortOpenInterest` → `openInterestShort`
- `indexTokenPrice` field REMOVED — uses CoinGecko fallback
**Stats**: 58 markets with >$10K OI, ~$70M total OI.

### 24. gTrade (Gains Network)

**Type**: DEX (Arbitrum) | **Data**: Funding, OI (NO live Tickers) | **Status**: Active

| Endpoint | URL | Method |
|----------|-----|--------|
| Trading Variables | `https://backend-arbitrum.gains.trade/trading-variables` | GET |

**Symbol Format**: `pair.from` field. Handles expiry suffixes (INTC_24_5 → INTC).
**Response Size**: ~2MB endpoint.
**Funding Interval**: 8h (velocity-based model, normalized from continuous)

**Funding Calculation** (extremely complex):
```
Velocity = skew_coefficient * net_exposure
Rate = velocity * time_elapsed
Capped by absoluteRatePerSecondCap
Converted to 8h percentage
```
Supports V1 and V2 borrowing models.

**Precision**: All values use custom per-field precision constants (15+ constants).
**Special**: 200+ delisted pairs filtered. Asset class grouping by group index. Prices from DB history merge (no live price endpoint).
**Known API Changes (Mar 2026)**:
- `pairDepths`, `pairSkewDepths`, `prices`, `pairOis` REMOVED from top level
- New: `pairInfos.pairDepthBands` (packed BigInt slot encoding)
- New: `oiWindows` (windowed OI)
- Pairs array still works (452 pairs, BTC=index 0)
- CoinGecko fallback needed for prices
