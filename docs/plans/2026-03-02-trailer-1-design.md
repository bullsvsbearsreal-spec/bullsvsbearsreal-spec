## Post-Production Pipeline

### Step 1: Asset Generation
1. Generate 4 reference images per character with Soul
2. Create characters with `create_character`
3. Wait for character processing (poll status)
4. Generate all 14 scene images (batch of 4 each = 56 images total)
5. Select best image per shot
6. Generate 12 DoP video clips from selected images
7. Record/generate voiceover audio (WAV format)
8. Generate 2 Speak v2 talking head clips

### Step 2: Assembly
1. Import all clips into video editor
2. Arrange in shot order
3. Trim/adjust timing to hit 60 seconds
4. Add crossfade transitions between shots (0.3-0.5s)
5. Add hard cuts on beat drops (shots 6, 9, 14)

### Step 3: Text & Graphics
1. Add text overlays per shot list
2. Use Orbitron font (matches existing InfoHub brand)
3. Gold (#FFA500) on dark backgrounds
4. Glitch effect on "DATA IS EVERYWHERE"
5. Slam/scale effect on feature names
6. "SOMETHING IS COMING" — massive scale with glow

### Step 4: Audio
1. Layer cinematic music track (dark electronic, 75 BPM)
2. Sync beats to shot transitions
3. Mix in voiceover at shots 11-12
4. Add SFX: whooshes, impacts, risers, glass breaks
5. Master to -14 LUFS for Twitter

### Step 5: Export
- Format: MP4 H.264
- Resolution: 1920x1080
- Frame rate: 30fps
- Bitrate: 8-10 Mbps
- Duration: exactly 60 seconds
- Thumbnail: Frame from Shot 15 (logo reveal)
